## 컬러 사용 가이드

### Primary — `#F2C4B0` (따뜻한 피치)
- 페이지 배경, 히어로 섹션 배경에 사용
- 전체 톤을 따뜻하고 차분하게 유지하는 베이스 컬러

### Secondary — `#C4B4BE` (소프트 모브 그레이)
- 카드 배경, 섹션 구분선, 그림자 효과에 사용
- Primary와 조화를 이루며 깊이감을 표현

### Accent — `#E8A882` / `#D4948A` (진한 피치)
- 버튼, 강조 텍스트, 호버 효과에 사용
- 포인트 컬러로 적절히 사용하여 시선 유도

### Text — `#3D2F35` / `#6B5B63` / `#B8A8B2`
- `--color-text-primary`: 제목, 본문 주요 텍스트
- `--color-text-secondary`: 부제목, 설명 텍스트
- `--color-text-muted`: 날짜, 태그, 보조 정보

### Interactive
- 버튼 기본: `#E8A882` → 호버: `#D4948A`
- 링크 기본: `#6B5B63` → 호버: `#3D2F35`

## 반응형 고려사항

### 다크모드 대응

```css
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg-primary: #2C2228;
    --color-bg-secondary: #3A2E33;
    --color-text-primary: #F2C4B0;
    --color-text-secondary: #C4B4BE;
    --color-text-muted: #7A6A70;
    --color-button-primary: #E8A882;
    --color-button-hover: #F2C4B0;
  }
}
```

### 화면 크기별 고려사항
- **모바일**: 배경 그라데이션 단순화, 텍스트 대비 강화
- **태블릿**: 카드 레이아웃에 Secondary 컬러 활용
- **데스크탑**: 풀 그라데이션 배경 및 섀도우 효과 적극 활용

### 접근성 (Accessibility)
- `--color-text-primary` (`#3D2F35`) + `--color-bg-primary` (`#F2C4B0`) → 대비비 약 **5.2:1** (WCAG AA 기준 충족)
- 버튼 텍스트는 반드시 `--color-text-primary` 사용 권장
